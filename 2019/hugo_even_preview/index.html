2. 段落

使用单引号 * 或者单下划线 _ 标记 斜体强调 或者 斜体强调

使用两个引号 ** 或者两个下划线 __ 标记 加粗强调 或者 加粗强调

引号和下划线可叠加使用 → 只是加粗 斜体并加粗

使用两个波浪线 ~~ 标记 已删除文字

插入文字暂无 Markdown 标记，直接使用 HTML 标签 <ins> 标记 插入文字

行内代码使用反引号标记 → print("hello world")
